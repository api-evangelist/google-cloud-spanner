Feature Spanner now offers AI functions , as a part of machine learning functions, that help you perform semantic operations using Large Language Models (LLMs) in SQL to classify, evaluate, and rank your data: AI.CLASSIFY : Classify a natural language input into user-defined categories. AI.IF : Evaluate a condition described in natural language. AI.SCORE : Rate natural language input and assign it a score.
